# Contributing

Thank you for your interest in this project! Contributions — especially independent replications — are welcome.

## How to Contribute

1. **Open an issue** describing what you'd like to change or report
2. **Fork the repository** and create a branch (`feat/your-change`)
3. **Make your changes** with clear commit messages
4. **Submit a pull request** referencing the issue

## Contribution Types

- **Reproduction reports** — share your independent replication attempts and results
- **Bug reports** — unexpected pipeline behaviour or numerical discrepancies
- **Documentation improvements** — always welcome
- **Extended analyses** — additional sensitivity analyses or alternative uncertainty specifications

## Code Style

- Python code follows PEP 8
- Use type hints where practical
- Keep the codebase minimal — this repo is intentionally self-contained
