# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability related to this repository:

1. Do not open a public issue with exploit details.
2. Email kareem.soliman@outlook.com.au with a clear reproduction and impact description.
3. Allow reasonable time for investigation and remediation before public disclosure.

## Scope

This policy applies to the code and repository infrastructure for this project.
It does not cover vulnerabilities in third-party dependencies themselves; report
those to their maintainers.
